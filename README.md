# Book Reader Telegram Mini App

Приложение для чтения книг с элементами социальной сети, работающее как Telegram Mini App.

## Технологии

- **Frontend**: React, React Router, Context API
- **Backend**: Node.js, Express, MongoDB
- **Деплой**: Vercel (frontend), Render (backend), MongoDB Atlas

## Функциональность

- Аутентификация через Telegram
- Каталог книг с поиском и фильтрацией
- Детальная страница книги с рейтингом и комментариями
- Читалка с поддержкой закладок
- Профиль пользователя со статистикой чтения
- Система прогресса чтения
